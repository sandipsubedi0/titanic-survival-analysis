# Titanic Survival Analysis

**Tools:** Python, Pandas, NumPy, Matplotlib, Seaborn  
**Dataset:** Titanic - Machine Learning from Disaster (Kaggle)  
**Source:** [Kaggle Titanic Dataset](https://www.kaggle.com/c/titanic/data)

---

## Project Overview

This is my second data analysis project. I analyzed the famous Titanic passenger dataset to uncover the key factors that determined survival — including gender, passenger class, age, and fare paid.

The project follows a structured 5-phase approach: data loading → exploration → cleaning → visualization → findings.

## Key Findings

- Only **38%** of passengers survived the disaster
- **Female passengers** had a ~74% survival rate vs ~19% for males
- **1st class passengers** had a ~63% survival rate vs ~24% for 3rd class
- **1st class females** had the highest survival rate (~97%)
- **3rd class males** had the lowest survival rate (~14%)
- Most passengers were aged **20–40 years old**
- Fare distribution was heavily skewed — most paid low fares, a few paid extremely high amounts
